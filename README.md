### Description

SplashGallery on Compose </br>
The app client with Image library using Unsplash api

### Tools

- JetpackCompose
- MVVM + MVI
- Dagger (DI)
- Ktor
- Coroutines / Flow
- Jetpack paging library
- Jetpack compose navigation
- Coil
- Material design 3 (Material YOU)
