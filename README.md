### Description

SplashGallery on Compose </br>
The app client with Image library using Unsplash api In order with dark theme support

### Tools

- JetpackCompose
- MVVM
- Dagger 2
- Retrofit 2
- Coroutines / Flow
- Jetpack pagination 3 / jetpack compose pagination
- Jetpack compose navigation
- ViewPager2
- Coil
